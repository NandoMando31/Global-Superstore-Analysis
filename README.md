# Global Superstore: Strategic Sales & Profitability Analysis 

##  Project Overview
This project presents a comprehensive strategic analysis of **Global Superstore** operations (2014-2017). Using advanced Excel tools, I transformed over 50,000 records into a decision-making system that identifies key markets, seasonal patterns, and profitability risks.

##  Interactive Dashboard
The core of this project is an interactive dashboard designed with a professional UI/UX focus. It features dynamic slicers for **Category**, **Segment**, and **Region**, allowing for real-time data exploration.

##  Key Business Insights

### 1. Market Concentration (Pareto Principle) 
* **Finding:** 80% of total revenue is generated by only 15 states. 
* **Insight:** Marketing and logistics efforts should be prioritized in these high-velocity zones to maximize ROI. **Arizona** was identified as the critical threshold state.

### 2. Seasonal Inventory Strategy (Q4 Focus) 
* **Trend:** Recurring demand spikes are observed every September and November.
* **Action:** It is recommended to start inventory ramp-up in **August** to prevent stockouts during the year-end peak.

### 3. Profitability Risk Management 
* **Risk Detected:** A significant portion of net profit depends on a single product: the **Canon imageCLASS 2200** copier (+$25k profit).
* **Recommendation:** Portfolio diversification of high-margin products is necessary to reduce financial vulnerability to supply chain shifts.

##  Methodology & Tools
* **Data Cleaning:** Used **Power Query** for data normalization, handling null values, and date formatting.
* **Data Modeling:** Developed complex **Pivot Tables** to segment sales by geography, time, and category.
* **Advanced Analysis:** Applied the **80/20 Rule (Pareto)** for strategic prioritization.
* **Visualization:** Designed an Excel interface using **Dynamic Charts** and a corporate color palette for executive reporting.
